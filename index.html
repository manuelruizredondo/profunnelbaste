0. TV Escaparate — Digital Signage Dental
El primer punto de contacto con el paciente: la calle
🚨

Máxima prioridad: La pantalla del escaparate es el único canal de marketing que funciona las 24 horas sin coste por impresión. Una TV bien programada convierte transeúntes en pacientes. Este es el primer activo a implementar, antes que cualquier campaña digital.

🎯

Por qué la TV del escaparate es prioritaria

  • Impacto visual inmediato a personas que pasan por delante
  • Funciona 24/7 sin intervención del equipo
  • Coste de producción amortizado en semanas
  • Transmite profesionalidad y modernidad instantáneamente
  • Puede incluir QR code para reserva inmediata de cita
  • Diferencia visualmente la clínica de la competencia
📐

Configuración técnica recomendada

  • Pantalla: 55"–65" Full HD o 4K, brillo ≥ 500 nits (uso vitrina)
  • Orientación: Horizontal (16:9) o Vertical (9:16) según espacio
  • Reproductor: Amazon Fire Stick 4K o Raspberry Pi 4
  • Gestión remota: Rise Vision o Screenly OSE (gratuito)
  • Conexión: WiFi o cable Ethernet para actualizaciones remotas
  • Protección: Pantalla de cristal anti-reflejo recomendada

Python — Generador de Copy para Spot de TV (GPT-4o) 
import openai

def generar_spot_tv(
    tratamiento: str,
    duracion_seg: int = 15,
    oferta: str = "",
    orientacion: str = "horizontal"
) -> dict:

    palabras_max = duracion_seg * 3  # ~3 palabras/segundo leídas en pantalla

    prompt = f"""
Crea el contenido textual para un spot de TV digital de escaparate de clínica dental.

Parámetros:
- Tratamiento: {tratamiento}
- Duración: {duracion_seg} segundos (máximo {palabras_max} palabras visibles)
- Oferta especial: {oferta if oferta else "ninguna"}
- Formato pantalla: {orientacion}

Devuelve JSON con esta estructura exacta:
{{
  "titular": "Texto grande, máximo 5 palabras, impacto máximo",
  "subtitulo": "Beneficio principal, máximo 8 palabras",
  "cuerpo": "Detalle opcional, máximo 12 palabras",
  "cta": "Llamada a la acción, máximo 4 palabras",
  "qr_texto": "Texto junto al QR, máximo 5 palabras",
  "color_fondo_sugerido": "#hexcolor",
  "emoji_decorativo": "1 emoji relevante"
}}
Tono: Profesional, confianza, urgencia suave. Español neutro.
"""

    response = openai.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "user", "content": prompt}],
        response_format={"type": "json_object"},
        temperature=0.6
    )

    spot_data = response.choices[0].message.content
    return spot_data

Python — Chatbot Handler (Make / n8n webhook) 
# Sistema prompt para el chatbot dental IA
SYSTEM_PROMPT = """
Eres el asistente virtual de Clínica Dental [NOMBRE].
Tu objetivo es:
1. Saludar calurosamente y preguntar en qué puedes ayudar
2. Identificar el tratamiento de interés (ortodoncia, implantes, blanqueamiento, etc.)
3. Recopilar nombre completo y método de contacto
4. Ofrecer disponibilidad de cita y redirigir al booking
5. Nunca diagnosticar. Si preguntan por síntomas, remitir al doctor.

Tono: cercano, profesional, en español. Respuestas máximo 3 frases.
"""

import openai, json

def process_lead_message(user_message: str, conversation_history: list) -> dict:
    messages = [
        {"role": "system", "content": SYSTEM_PROMPT},
        *conversation_history,
        {"role": "user", "content": user_message}
    ]

    response = openai.chat.completions.create(
        model="gpt-4o",
        messages=messages,
        temperature=0.4,
        max_tokens=200,
        response_format={"type": "json_object"},
        functions=[{
            "name": "capture_lead_data",
            "parameters": {
                "type": "object",
                "properties": {
                    "nombre": {"type": "string"},
                    "tratamiento_interes": {"type": "string"},
                    "telefono": {"type": "string"},
                    "lead_score": {"type": "number", "description": "0-100"}
                }
            }
        }]
    )
    return response.choices[0].message

JavaScript — HubSpot Contact Upsert via API 
const upsertDentalLead = async (leadData) => {
  const { nombre, email, telefono, tratamiento, leadScore } = leadData;

  const payload = {
    properties: {
      firstname: nombre.split(' ')[0],
      lastname: nombre.split(' ').slice(1).join(' '),
      email: email,
      phone: telefono,
      tratamiento_interes: tratamiento,
      hs_lead_status: 'NEW',
      lead_score_ia: leadScore,
      fuente_captacion: 'chatbot_web'
    }
  };

  const response = await fetch(
    'https://api.hubapi.com/crm/v3/objects/contacts/upsert',
    {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${process.env.HUBSPOT_TOKEN}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify(payload)
    }
  );

  if (response.ok) {
    // Disparar secuencia de nurturing en ActiveCampaign
    await triggerNurturingSequence(email, tratamiento);
  }
};

Python — Generador de Guión Dental con GPT-4o 
import openai

def generar_guion_dental(tratamiento: str, duracion_min: int = 3) -> str:
    palabras_objetivo = duracion_min * 130  # ~130 palabras/minuto hablado

    prompt = f"""
Eres un guionista especializado en salud dental para redes sociales.
Crea un guión de vídeo de {duracion_min} minutos (~{palabras_objetivo} palabras)
sobre: {tratamiento}

Estructura obligatoria:
[HOOK - 10 seg] Pregunta o dato impactante que engancha
[PROBLEMA - 30 seg] Dolor o preocupación del paciente
[SOLUCIÓN - 90 seg] El tratamiento explicado sencillamente
[PRUEBA SOCIAL - 30 seg] Resultado típico / testimonial genérico
[CTA - 20 seg] Llamada a la acción: "Agenda tu primera visita gratuita"

Tono: Profesional, cercano, sin tecnicismos. Para pacientes 30–55 años.
Incluye [PAUSA] donde aplique y [B-ROLL: descripción] para el editor.
    """

    response = openai.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "user", "content": prompt}],
        temperature=0.7,
        max_tokens=1500
    )

    guion = response.choices[0].message.content

    # Generar también metadatos SEO para YouTube
    seo = generar_seo_youtube(guion, tratamiento)

    return {"guion": guion, "seo": seo}

# Uso: generar_guion_dental("Implantes dentales", duracion_min=4)
